# 一、项目搭建
  ## 1. 准备： yarn 安装
  ## 2. 创建项目 yarn create umi。 选择所需依赖，dva + antd + umi
  ## 3. 启动项目 yarn start
  ### 注：以上部分与下一部分均是根据antd官网提供,项目其他配置请参考
 # 二、编码便捷推荐
  ## 1. 用命令创建路由、组件
    ### 1). 安装 npx,   yarn global add npx
    ### 2). 创建路由   npx umi g + 目录地址
	### 如：npx umi g page products，创建文件在 src/pages/products.js，…
